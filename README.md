# Founder's Run — VCM Startup-Simulation

Startup-Simulation für den VCM-Infostand auf der Startup Contacts.  
~3 Minuten Spielzeit: Entscheidungen treffen, Founder-Typ erhalten, Rückblick mit Alternativen.

## App öffnen

**[founders-run-sepia.vercel.app](https://founders-run-sepia.vercel.app)**

## Inhalte bearbeiten

Alle Texte, Fragen und Antworten stehen als einfache Tabellen in [`startup-simulation/content/`](startup-simulation/content/).  
Jede Datei lässt sich direkt im GitHub-Webeditor bearbeiten — kein Code, kein lokales Setup nötig.

Anleitung: [`docs/CMS-HANDBUCH.md`](docs/CMS-HANDBUCH.md)
