# Archivierter Folgeprompt

Dieser Folgeprompt ist umgesetzt beziehungsweise abgelöst. Neue Änderungen
werden direkt in der Next.js-App unter `startup-simulation/` geplant und
implementiert.

Aktuelle Leitplanken:

1. Mira bleibt das fiktive Startup.
2. Geld wird in Euro gerechnet und angezeigt.
3. Pro Durchlauf: 5 Entscheidungen, 1 VCM-Event, 1 Markt-Event, 1 Verteil-Runde.
4. Ergebnis kann als Story-Grafik geteilt werden.
5. Die App bleibt standalone und Vercel-basiert.
